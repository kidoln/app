MeshWork 操作手册

MeshWork 是围绕笔记、讨论、AI 与 Git 共享构建的创作工作台。 本手册将帮助你系统性地了解 MeshWork 的使用方式。

核心概念

∞ 工作台
工作台是用户进入产品后看到的主使用环境，负责组合私有知识库和多个共享空间。它是一个产品和交互层概念，不是固定磁盘目录，也不是 Git 仓库。
📁 私有知识库
属于当前用户自己的数据边界，存放个人长期知识、不应自动进入团队共享的内容。它独立存在，不从属于任何共享空间。
👥 共享空间
团队或项目级的数据边界。每个共享空间包含公共知识库、讨论区、共享附件，并可选拥有自己的 Git 历史。
💬 讨论区
共享空间内的异步讨论空间，支持话题管理、@ 提及、# 文档引用、/ 命令，以及 AI 智能体协作。
🤖 AI 智能体
MeshWork 中的 AI 是参与者而非自动替代者。它可以提出建议，但不会未经确认直接修改你的文档。

产品架构与知识流转

MeshWork 采用"私有优先、渐进共享"的设计理念，知识从个人创作流向团队协作，最终沉淀为可复用的长期知识。

知识流转路径：在私有知识库中进行个人创作和积累，AI 作为助手提供智能辅助。需要团队协同时，你可以直接将内容共享到共享空间：在讨论区进行快速交流，有价值的内容进一步沉淀到公共知识库成为长期知识。AI 在各个环节提供辅助，但不会自动共享你的私有内容。
