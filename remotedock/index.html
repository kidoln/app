FAQ

常见问题

Remote Dock 是免费的吗？

Mac 版本：完全免费
iOS 版本：基础功能免费，高级功能需一次性购买

需要互联网连接吗？

不需要。只需本地网络或 Apple 近距连接即可使用。

数据安全吗？

完全安全。所有数据仅在本地设备间传输，不上传云端。

点击应用图标后没有立即响应？

这通常是由于本地网络干扰造成的。请稍等片刻，或尝试重启应用。如果问题持续，请确保两台设备在同一网络下。

当前功能有哪些局限？

当桌面使用多个空间（Space）时，应用启动可能无法完全符合预期。这是由于苹果提供的 API 限制所致。

我需要更新应用吗？

强烈建议您随时更新到最新版本。新版本不仅带来新功能，还能修复已知问题，提升使用体验。请定期关注本站主页获取最新动态。

FAQ

Frequently Asked Questions

Is Remote Dock free?

Mac version: Completely free
iOS version: Basic features free, one-time purchase for advanced features

Need internet connection?

No. Just local network or Apple Peer-to-Peer connection.

Is my data secure?

Completely secure. All data is transferred locally between devices, never uploaded to the cloud.

App doesn't respond immediately when tapping icon?

This is usually caused by local network interference. Please wait a moment or try restarting the app. If the issue persists, ensure both devices are on the same network.

What are the current limitations?

When using multiple desktop Spaces, app launching may not always work as expected. This is due to limitations in Apple's provided APIs.

Should I update the app?

We strongly recommend keeping your app updated to the latest version. New versions bring new features and fix known issues, improving your overall experience. Please check this website regularly for the latest updates.
